# Selenium Hybrid Test Automation Framework

## Overview
A robust test automation framework built with Java, Selenium WebDriver, and TestNG using Page Object Model (POM).

## Tech Stack
- Java 11+
- Selenium WebDriver
- TestNG
- Maven
- Extent Reports
- Log4j

## How to Run Tests
```bash
mvn clean test
```

## Features
- Page Object Model (POM) design pattern
- Cross-browser testing (Chrome, Firefox, Edge)
- Data-driven testing with TestNG
- Extent Reports with screenshots on failure
- Retry failed tests automatically
- Configurable properties file
